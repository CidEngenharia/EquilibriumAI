# Equilibrium – Sistema de Decisão Inteligente

Bem-vindo ao Equilibrium, um SaaS focado em ajudar você a tomar decisões mais equilibradas usando uma combinação de filosofias milenares e psicologia moderna.

## 🚀 Como rodar o projeto

### 1. Backend
```bash
cd backend
npm install
npm run dev
```
O backend rodará em `http://localhost:3001`.

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```
O frontend rodará em `http://localhost:3000`.

## 🛠️ Tecnologias
- **Frontend**: React + Tailwind CSS + Vite
- **Backend**: Node.js + Express
- **IA**: Google Gemini 1.5 Flash

## 🧠 Filosofia do Sistema
O assistente analisa suas decisões sob as lentes de:
- **Ikigai & Kaizen**: Propósito e melhoria contínua.
- **Estoicismo**: Foco no que você pode controlar.
- **Psicologia**: Vieses e heurísticas.
- **Valores**: Ética e integridade.
